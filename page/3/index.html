JAVA基础知识复习笔记

发表于 2015-10-22 | 更新于 2016-07-11 | 分类于 JAVA
  1. java程序初始化:静态代码快优先与非静态代码块(优先于构造函数),父类优先于子类进行初始化
  2. 没有声明任何方法的接口为标识接口,用来判断某个类属于特定的类型,用instanceof判断是否是某个接口,类或者抽象类的实现
  3. JAVA浅复制和深复制:都需要实现Cloneable标识接口.当一个类只包含基本数据类型是,使用浅复制即可复制该对象,但是包含其他对象是,需要在clone方法中对每个属性进行深复制.
  4. JAVA创建对象的四种方法:new , 反射 , 类的复制 , 反序列化
  5. JAVA实现回调函数的方法:通过定义一个接口,然后在接口中声明要调用的方法,接着实现该接口,在需要调用的地方把该类接口类型(相当于c++的函数指针)作为参数传进去,实际使用是传入该接口不同的实现类的对象即可.
  6. JAVA中多态的机制:编译时多态(通过方法的重载实现),运行时多态(通过方法的重写实现)
  7. native关键字:native关键字说明其修饰的方法是一个原生态方法，方法对应的实现不是在当前文件，而是在用其他语言（如C和C++）实现的文件中。Java语言本身不能对操作系统底层进行访问和操作，但是可以通过JNI接口调用其他语言来实现对底层的访问。
  8. 当子类需要显式的调用父类的构造函数时,super()必须是第一条语句.
  9. JAVA跳出多层循环:在需要跳出的地方设置一个标识,然后在break(break out;此处out即为标识,类似于goto语句的使用)语句后天添加标识即可.
  10. final是指引用不可变,但是对于引用的对象可以改变
  11. Math类中ceil,round,floor的区别
  12. new String(“abc”)创建了几个对象
  13. 对象默认的equals方法是比较的引用
  14. StringBuilder的效率高于StringBuffer,但是其不是线程安全的
  15. try catch中如果有return语句,但是依旧会执行finally里面的语句,但是如果finally里面有return语句的话,会覆盖其他return语句
  16. 异常:异常分为运行时异常和普通异常,分别为Error和Exception类,均继承自Throwable类,运行时异常会导致程序的终止执行,而普通异常用户可以自己处理
  17. JAVA中的两种IO流,字节流(InputStream和OutputStream)和字符流(Reader和Writer,其他流继承自这两种类从而实现不同的功能
  18. JAVA集合框架:List,Set,Map,迭代器Iterator,

ubuntu下ftp服务器搭建

发表于 2015-08-23 | 更新于 2016-04-03 | 分类于 服务器

apt-get install vsftpd 安装ftp服务器端
vim /etc/vsftpd.conf 编辑ftp服务器的配置

如果设置了*anonymous_enable=YES*则表示可以匿名登陆

启用这兩项的话，根目录要设置为不可写

添加用户

由于用户组在安装的时候已经默认添加了用户组ftp，所以直接添加用户即可

useradd kangbiao -g ftp -d /var/www/heyunjiang -s /sbin/nologin
passwd kangbiao 设置新添加用户的密码

还需要在编辑/etc/shells文件，在末行加入/sbin/nologin

新建的目录要将权限设为777才可进行ftp访问（很笨的做法，只需要保证ftp用户组的权限为7就行）

每次编辑ftp配置后记得 service vsftpd restart重启ftp服务器使更改生效

crontab在Ubuntu下的使用

发表于 2015-07-06 | 更新于 2016-04-03 | 分类于 服务器
  1. crontab配置文件
  2. crontab -e 编辑当前用户的任务列表
  3. crontab -l 查看计划任务列表
    配置文件格式
        • * command 号分别对应 分钟0-59 小时0-23 日期1-31 月份1-12 星期0-7 中间以空格分开

例子：

  1. 每晚21:31重启Apache服务器

  2. 每月的1,22,24,日的3:30重启Apache

  3. 每月的20-30号的3:30重启Apache

  4. 每隔两分钟重启Apache

  5. 晚上11点到早上7点，每隔一小时重启Apache

  6. 每天18点到23点每隔30分钟重启Apache

含义
a,b,c逗号分隔表示a或b或c三个时间点执行任务
a-b表示a到b这个时间段执行任务
*/a表示每a分钟或者小时或者天或者月或者星期执行任务

  1. 切换用户为root：su root(如果没有为root设置密码，就需要sudo passwd为root设置密码，不然不能切换)。

  2. 安装服务器组件：apt-get install apache2 php5 mysql-server php5-mysql。

  3. 运行cat /etc/apache2/mod–enabled/php5.load命令查看Apache和php解析器是否关联。

  4. 运行cat /etc/php5/mods-available/mysql.ini命令查看php解析器和mysql组件是否关联成功。如果提示没有那个文件或目录就用ls命令找到mysql.ini文件。

  5. 重启Apache和mysql服务：service apache2 restart;service mysql restart。

  1. 测试安装是否成功，打开vim输入下面的代码 vim info.php（需要注意的是Apache的默认根目录是www/html，略坑。。）

发表于 2015-02-05 | 更新于 2016-04-03 | 分类于 SQL&NoSQL

基本语法：

mongodb内置的帮助功能非常实用。需要查看哪一级操作的帮助，直接xxx.help()即可，便会列出所有可用的操作.后面会有英文说明，看说明就知道对应的方法是干什么用的。

概念：

mongodb存储结构：

db(相当于关系型数据库中的数据库)：共用同一个索引的collections的集合
collection(相当于关系型数据库中的表)：共用一个索引的documents的集合。
document(相当于关系性数据库中的一行记录，即一个实体对象)：共用一个索引的对象属性的集合。
可见mongodb中把关系型数据库中的包含存储关系变为索引式的存储方式。以二进制表示的json数据格式（BSON）存储数据，用索引加快读取速度，所以mongodb这类非关系型数据库的读取速度会比关系型数据库快很多。
mongodb中的索引：为了提高查询和更新速度，mongodb使用了间接索引，应用程序可以把部分collections存储在一个读取速度很快的数据结构中方便读取

CURD操作：

查询操作

原理：在mongodb中，一个查询的对象是一系列共用一个索引的文档的集合，也就是一个collection。查询的时候根据查询的准则和条件返回documents结果。
例子：db.test.find({age:{$gt:19}}).sort({age:1}) 查询test集合中年龄大于19的对象的集合，并且按照年龄从小到大排序。

db.table1.find({name:’kangbiao’},{id:1})默认会选择_id字段，如果不需要则添加_id:0排除此字段

db.table1.find({name:’kangbiao’},{_id:1})

db.table1.find({name:’kangbiao’},{_id:0})

插入操作
例子：

db.test.insert({age:19,name:’xiaoming’,pwd:’adgsfsdsdfs’})

将小明这个对象插入到test集合中。
